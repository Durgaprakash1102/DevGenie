DevGenie

“AI-powered DevOps Automation Assistant”

This web application was built using Ollama with the LLaMA 3.2 (3B parameter) open-source LLM model installed locally. It generates Dockerfile, Jenkinsfile, Terraform, Ansible, Kubernetes, and GitHub Actions configurations.

The goal is to showcase a blend of development skills, ongoing DevOps learning, and AI-assisted tooling.

Since this web app uses a smaller model, review the output before production use.

With larger models and refined prompts, this concept can evolve into a powerful DevOps AI assistant.
